# sfPaint
An attempt to make a paint clone in sfml( also while using Imgui)
## Prerequisites 
Cmake(>=3.0)\
Ccache(if on linux)
## Install
1. Clone the repo
```
git clone https://github.com/o-dka/sfPaint.git
```
2. Run 
```./cmake_cache.sh```
3. Build with cmake 
```
cmake --build build
```
4.Run the program!
```
./build/sfPaint
```
